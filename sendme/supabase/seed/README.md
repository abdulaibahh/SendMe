# SendMe Seed Data

The canonical seed migration is `../migrations/004_seed_data.sql`.

It includes Sierra Leone market context:

- Categories: Vegetables, Fruits, Rice, Meat, Fish, Cooking Ingredients, Household Items, Beverages.
- Products priced in SLE with market-friendly units.
- Service areas: Lumley, Aberdeen, Wilberforce, Congo Cross, Hill Station, Kissy, Juba, Murray Town.
- Demo rider profiles that are not linked to authentication accounts until an admin links them.

Live payment merchant numbers and production support contacts are intentionally excluded.
